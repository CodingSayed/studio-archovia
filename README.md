# Studio Archovia MVP

Studio Archovia is an English-language magazine-style home base for a curated architecture and interior design publication.

## Current Direction

- Spacious luxury editorial layout
- Royal blue used as a restrained brand accent
- Founder credit: Gratto Grattjoes, Founder & Curator
- TikTok and LinkedIn growth focus
- Placeholder article, archive, contact, and email content
- Static MVP that can be published quickly

## Pages / Sections

- Home hero
- Featured editorial story
- Journal / latest selections
- Visual archive
- Interior design positioning section
- About / founder section
- Contact placeholders

## Later Features

- Admin dashboard or CMS
- Real article publishing flow
- Search and filtering
- Newsletter signup
- Submission page
- Saved posts
- Tumblr-style reposting or collections
- User accounts

## Run Locally

Open `index.html` directly in a browser.

For a local server:

```bash
npx serve .
```
